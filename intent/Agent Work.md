# Agent Work

Tags: #agent-work #execution

Agent work is attempted implementation, analysis, review, or maintenance performed by Cerberus or another agent.

Agent work is not accepted by default. It produces [[Evidence]], which supports a [[Decision]].

## Good Agent Work

- Starts from [[Intent]], [[Spec]], and known [[Constraints]].
- Produces inspectable [[Change]].
- Leaves enough [[Evidence]] to review behavior.
- Avoids unrequested broad refactors.
