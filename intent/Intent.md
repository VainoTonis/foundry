# Intent

Tags: #intent #north-star

Intent is the user's durable will for a project: goals, constraints, preferences, unanswered questions, and accepted tradeoffs.

Intent is not a prompt. A prompt is temporary. Intent should survive conversations, agent runs, failed attempts, and UI changes.

## Foundry Loop

[[Intent]] becomes [[Spec]].

[[Spec]] becomes [[Phase]].

[[Phase]] becomes [[Agent Work]].

[[Agent Work]] produces [[Evidence]].

[[Evidence]] supports [[Decision]].

[[Decision]] updates [[Intent]].

## Why It Matters

Without durable intent, every agent run starts from scattered context. With durable intent, Foundry can generate better specs, review work against real goals, and explain why changes were accepted.

## Boundaries

Intent should describe why and what. Specs should describe what to build next. Phases should describe bounded execution steps.
